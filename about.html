Sobre el Proyecto

Este es nuestro primer proyecto dentro del curso de Programador Web Full Stack en la UTN. Representa el inicio de nuestro camino en el desarrollo web y busca poner en práctica los conceptos y herramientas que fuimos adquiriendo durante la cursada.

Más allá del producto final, lo que este proyecto refleja es nuestro proceso de aprendizaje: la capacidad de investigar, resolver problemas, trabajar con código limpio y aplicar buenas prácticas en cada etapa del desarrollo.

A quién está dirigido

Este trabajo está pensado principalmente para nuestros docentes, quienes evaluarán nuestro desempeño y progreso en la materia. A través de este proyecto buscamos mostrar no solo lo aprendido, sino también nuestro compromiso y esfuerzo en el desarrollo web.

Al mismo tiempo, lo vemos como una oportunidad de generar un producto funcional y visualmente atractivo, que podría ser la base para futuros proyectos más complejos.

Tecnologías Utilizadas

Para la realización de este proyecto implementamos distintas tecnologías web:

  • HTML para la estructura base del sitio.
  • CSS para el diseño y la presentación.
  • JavaScript para la lógica y la interactividad.
  • React para la construcción de una interfaz modular y dinámica.
  • APIs para la integración de datos externos en tiempo real.

Estas herramientas nos permitieron dar forma a una aplicación moderna, adaptable a distintos dispositivos y alineada con las prácticas actuales del desarrollo web.
